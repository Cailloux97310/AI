# SECTION 3 — Encodage par substitution v7 (Excelsior_XW72)

## Format de reconnaissance

```
Ligne 1   : Excelsior_XW72
Corps     : tokens de 6 caractères séparés par espaces
[Rn]      : marqueur de rotation explicite (émis tous les 50 tokens)
WORDSEP   : séparateur de mots
NEWLIN    : saut de ligne
[CRC:XXXX]: checksum CRC16 optionnel en fin de ligne encodée
```

---

## Propriétés de bijection

- Chaque caractère ASCII imprimable (codes 32–126, soit **95 valeurs**) est mappé
  à exactement **un token de 6 caractères alphanumériques**
- La table est strictement inversible : chaque token correspond à un seul caractère
- **Pas de segmentation syllabique** : encodage octet par octet, sans ambiguïté
- La rotation `[Rn]` est un décalage additif **modulo 95** sur l'index du caractère,
  appliqué AVANT lookup : `token = TABLE[(ord(c) - 32 + rotation) % 95]`
- La rotation est portée par sa valeur explicite dans `[Rn]` → pas d'ambiguïté
  même en cas de lecture partielle du fichier

---

## Algorithme d'encodage

```
1. Initialiser rotation = 0, compteur_tokens = 0
2. Pour chaque ligne du texte clair :
   a. Pour chaque mot (séparé par espaces) :
      - Pour chaque caractère c du mot :
        i.  idx = (ord(c) - 32 + rotation) % 95
        ii. Émettre TABLE[idx]
        iii.compteur_tokens += 1
        iv. Si compteur_tokens % 50 == 0 :
              rotation = (rotation + 1) % 95
              Émettre [R{rotation}]
      - Émettre WORDSEP après chaque mot
   b. Émettre NEWLIN en fin de ligne
3. Émettre optionnellement [CRC:XXXX] en fin de chaque ligne encodée
```

## Algorithme de décodage

```
1. Initialiser rotation = 0, compteur_tokens = 0, texte = ""
2. Pour chaque token T dans le flux :
   - Si T == "WORDSEP" : ajouter ' ' au texte
   - Si T == "NEWLIN"  : ajouter '\n' au texte
   - Si T correspond à [Rn] : rotation = n (lire n explicitement)
   - Si T correspond à [CRC:XXXX] : vérifier CRC16 de la ligne, signaler erreur si KO
   - Sinon (token de données) :
       i.  idx = TABLE_INV[T]  # lookup inverse, O(1)
       ii. c = chr((idx - rotation + 95) % 95 + 32)
       iii.Ajouter c au texte
       iv. compteur_tokens += 1
       v.  Si compteur_tokens % 50 == 0 :
             rotation = (rotation + 1) % 95
             (le prochain token [Rn] dans le flux confirme la valeur — vérification)
```

---

## Garanties de bijection

| Propriété | Détail |
|---|---|
| Injection (encodage) | 95 caractères → 95 tokens distincts, pas de collision |
| Surjection (décodage) | TABLE_INV couvre exactement les 95 tokens émissibles |
| Rotation déterministe | Compteur synchrone encode/decode + valeur explicite [Rn] |
| Longueur fixe | Token = exactement 6 chars alphanumériques, impossible de confondre avec WORDSEP/NEWLIN/[Rn] |
| Intégrité | CRC16 (polynôme 0x1021) détecte toute corruption bit |

## Script Python de référence (encode / decode)

```python
#!/usr/bin/env python3
"""
Excelsior_XW72 — Codec de référence v7
Encodage/décodage bijectif garanti, sans ambiguïté syllabique.
Usage:
    python excelsior_xw72.py encode <input.txt> <output.txt>
    python excelsior_xw72.py decode <input.txt> <output.txt>
"""

import sys
import re

# ---------------------------------------------------------------------------
# Table de substitution v7 — 95 entrées
# Index 0 = ASCII 32 (espace), index 94 = ASCII 126 (~)
# ---------------------------------------------------------------------------
_TABLE_RAW = [
    "Xa0Qr1","Xa1Kp2","Xa2Lm3","Xa3Np4","Xa4Mp5",  # 32-36  : ' !"#$'
    "Xa5Rq6","Xa6Ts7","Xa7Uw8","Xa8Vx9","Xa9Wy0",  # 37-41  : '%&\'()'
    "Xb0Xz1","Xb1Ya2","Xb2Zb3","Xb3Ac4","Xb4Bd5",  # 42-46  : '*+,-.''
    "Xb5Ce6","Xb6Df7","Xb7Eg8","Xb8Fh9","Xb9Gi0",  # 47-51  : '/0123'
    "Xc0Hj1","Xc1Ik2","Xc2Jl3","Xc3Km4","Xc4Ln5",  # 52-56  : '45678'
    "Xc5Mo6","Xc6Np7","Xc7Oq8","Xc8Pr9","Xc9Qs0",  # 57-61  : '9:;<='
    "Xd0Rt1","Xd1Su2","Xd2Tv3","Xd3Uw4","Xd4Vx5",  # 62-66  : '>?@AB'
    "Xd5Wy6","Xd6Xz7","Xd7Ya8","Xd8Zb9","Xd9Ac0",  # 67-71  : 'CDEFG'
    "Xe0Bd1","Xe1Ce2","Xe2Df3","Xe3Eg4","Xe4Fh5",  # 72-76  : 'HIJKL'
    "Xe5Gi6","Xe6Hj7","Xe7Ik8","Xe8Jl9","Xe9Km0",  # 77-81  : 'MNOPQ'
    "Xf0Ln1","Xf1Mo2","Xf2Np3","Xf3Oq4","Xf4Pr5",  # 82-86  : 'RSTUV'
    "Xf5Qs6","Xf6Rt7","Xf7Su8","Xf8Tv9","Xf9Uw0",  # 87-91  : 'WXYZ['
    "Xg0Vx1","Xg1Wy2","Xg2Xz3","Xg3Ya4","Xg4Zb5",  # 92-96  : '\]^_`'
    "Xg5Ac6","Xg6Bd7","Xg7Ce8","Xg8Df9","Xg9Eg0",  # 97-101 : 'abcde'
    "Xh0Fh1","Xh1Gi2","Xh2Hj3","Xh3Ik4","Xh4Jl5",  # 102-106: 'fghij'
    "Xh5Km6","Xh6Ln7","Xh7Mo8","Xh8Np9","Xh9Oq0",  # 107-111: 'klmno'
    "Xi0Pr1","Xi1Qs2","Xi2Rt3","Xi3Su4","Xi4Tv5",  # 112-116: 'pqrst'
    "Xi5Uw6","Xi6Vx7","Xi7Wy8","Xi8Xz9","Xi9Ya0",  # 117-121: 'uvwxy'
    "Xj0Zb1","Xj1Ac2","Xj2Bd3","Xj3Ce4","Xj4Df5",  # 122-126: 'z{|}~'
]

# Vérification statique de bijection
assert len(_TABLE_RAW) == 95, "Table incomplète"
assert len(set(_TABLE_RAW)) == 95, "Collision détectée dans la table"

TABLE     = _TABLE_RAW                          # index → token
TABLE_INV = {tok: idx for idx, tok in enumerate(TABLE)}  # token → index

WORDSEP = "WORDSEP"
NEWLIN  = "NEWLIN"
ROTATION_RE = re.compile(r'^\[R(\d+)\]$')
CRC_RE      = re.compile(r'^\[CRC:([0-9A-Fa-f]{4})\]$')


# ---------------------------------------------------------------------------
# CRC16 (CCITT, polynôme 0x1021)
# ---------------------------------------------------------------------------
def crc16(data: str) -> int:
    crc = 0xFFFF
    for byte in data.encode('utf-8'):
        crc ^= byte << 8
        for _ in range(8):
            crc = ((crc << 1) ^ 0x1021) if (crc & 0x8000) else (crc << 1)
            crc &= 0xFFFF
    return crc


def crc16_hex(data: str) -> str:
    return f"{crc16(data):04X}"


# ---------------------------------------------------------------------------
# Encodage
# ---------------------------------------------------------------------------
def encode(text: str, add_crc: bool = True) -> str:
    """Encode un texte clair en format Excelsior_XW72."""
    output_lines = ["Excelsior_XW72"]
    rotation = 0
    counter  = 0

    for line in text.splitlines():
        tokens = []
        words  = line.split(' ')
        for w_idx, word in enumerate(words):
            for ch in word:
                code = ord(ch)
                if code < 32 or code > 126:
                    # Caractère hors plage : on ignore ou on substitue
                    continue
                idx = (code - 32 + rotation) % 95
                tokens.append(TABLE[idx])
                counter += 1
                if counter % 50 == 0:
                    rotation = (rotation + 1) % 95
                    tokens.append(f"[R{rotation}]")
            if w_idx < len(words) - 1:
                tokens.append(WORDSEP)
        tokens.append(NEWLIN)

        encoded_line = ' '.join(tokens)
        if add_crc:
            encoded_line += f" [CRC:{crc16_hex(line)}]"
        output_lines.append(encoded_line)

    return '\n'.join(output_lines)


# ---------------------------------------------------------------------------
# Décodage
# ---------------------------------------------------------------------------
def decode(encoded_text: str, check_crc: bool = True) -> str:
    """Décode un texte au format Excelsior_XW72 vers le texte clair."""
    lines = encoded_text.splitlines()
    if not lines or lines[0].strip() != "Excelsior_XW72":
        raise ValueError("En-tête Excelsior_XW72 manquant ou invalide")

    rotation = 0
    counter  = 0
    output_lines = []

    for line in lines[1:]:
        tokens = line.split()
        current_line = ""
        expected_crc = None

        # Extraire le CRC s'il est présent en fin de ligne
        if tokens and CRC_RE.match(tokens[-1]):
            m = CRC_RE.match(tokens[-1])
            expected_crc = m.group(1).upper()
            tokens = tokens[:-1]

        for tok in tokens:
            if tok == NEWLIN:
                break
            if tok == WORDSEP:
                current_line += ' '
                continue
            m = ROTATION_RE.match(tok)
            if m:
                rotation = int(m.group(1))
                continue
            # Token de données
            if tok not in TABLE_INV:
                raise ValueError(f"Token inconnu : '{tok}'")
            idx = TABLE_INV[tok]
            ch  = chr((idx - rotation + 95) % 95 + 32)
            current_line += ch
            counter += 1
            if counter % 50 == 0:
                rotation = (rotation + 1) % 95

        # Vérification CRC
        if check_crc and expected_crc is not None:
            actual_crc = crc16_hex(current_line)
            if actual_crc != expected_crc:
                raise ValueError(
                    f"CRC invalide sur la ligne décodée '{current_line}' "
                    f"(attendu {expected_crc}, obtenu {actual_crc})"
                )
        output_lines.append(current_line)

    return '\n'.join(output_lines)


# ---------------------------------------------------------------------------
# Validation interne — round-trip test
# ---------------------------------------------------------------------------
def self_test():
    samples = [
        "Bonjour le monde !",
        "Test 1-2-3 : encodage v7.",
        "Caractères spéciaux : @#$%^&*()_+-=[]{}|;':\",./<>?",
        "Ligne avec des chiffres : 0123456789",
        "abcdefghijklmnopqrstuvwxyz ABCDEFGHIJKLMNOPQRSTUVWXYZ",
        " ",   # espace seul
        "~",   # dernier caractère ASCII imprimable
    ]
    print("=== Auto-test Excelsior_XW72 ===")
    all_ok = True
    for s in samples:
        encoded = encode(s, add_crc=True)
        decoded = decode(encoded, check_crc=True)
        ok = (decoded.strip() == s)
        status = "OK" if ok else "FAIL"
        if not ok:
            all_ok = False
        print(f"  [{status}] '{s[:40]}{'...' if len(s) > 40 else ''}'")
        if not ok:
            print(f"         → obtenu : '{decoded}'")
    print("=== Résultat :", "TOUS OK" if all_ok else "ÉCHEC(S) DÉTECTÉ(S)", "===\n")
    return all_ok


# ---------------------------------------------------------------------------
# Point d'entrée CLI
# ---------------------------------------------------------------------------
if __name__ == "__main__":
    if len(sys.argv) == 1 or sys.argv[1] == "test":
        self_test()
        sys.exit(0)

    if len(sys.argv) != 4 or sys.argv[1] not in ("encode", "decode"):
        print(__doc__)
        sys.exit(1)

    mode    = sys.argv[1]
    in_path = sys.argv[2]
    out_path= sys.argv[3]

    with open(in_path, 'r', encoding='utf-8') as f:
        content = f.read()

    if mode == "encode":
        result = encode(content, add_crc=True)
    else:
        result = decode(content, check_crc=True)

    with open(out_path, 'w', encoding='utf-8') as f:
        f.write(result)

    print(f"{mode} terminé : '{in_path}' → '{out_path}'")
```

## Notes de sécurité

- La table v7 est **publique** dans ce fichier ; la confidentialité n'est pas l'objectif premier
- Pour un usage chiffré, remplacer la table par une **permutation secrète** des 95 tokens
  générée par un PRNG seedé avec une passphrase (non stockée ici)
- La rotation apporte une légère diffusion mais ne constitue pas un chiffrement fort
- Usage recommandé : **obfuscation** de documents sensibles en transit, pas chiffrement cryptographique
